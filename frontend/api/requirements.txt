 The  requirements.txt  file is used to specify the dependencies of the project. The  pip  tool can install all the dependencies listed in the  requirements.txt  file. 
 The  requirements.txt  file contains the following dependencies: 
 
 flask : The Flask web framework. 
 flask-cors : A Flask extension for handling Cross-Origin Resource Sharing (CORS). 
 python-dotenv : A Python library for reading environment variables from a  .env  file. 
 requests : A Python library for making HTTP requests. 
 
 Create a  .env  File 
 The  .env  file is used to store environment variables. The  python-dotenv  library is used to read the environment variables from the  .env  file.
